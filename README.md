# Installation

## Server

```sh
cd packages/server
npm install
npm run watch
```

## Client

```sh
cd packages/web
npm install
npm run dev
```

# Features

- Can download videos from magnet link
- Stream downloaded videos
- Realtime download progress for downloading videos
